Description
===========
Installs Sencha Touch

Requirements
------------
#### platform
- `OS X` - 10.6.8 or greater.

Attributes
----------

Usage
-----
#### sencha_touch::mac

Just include `sencha_touch:` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[sencha_touch::mac]"
  ]
}
```
