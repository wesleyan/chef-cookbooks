perkinelmer_chemdraw_pro Cookbook
===========================
This cookbook installs PerkinElmer ChemDraw Pro

Requirements
------------
OS X 10.6.8+

Attributes
----------

Usage
-----
Just include `perkinelmer_chemdraw_pro` in your node's `run_list`
