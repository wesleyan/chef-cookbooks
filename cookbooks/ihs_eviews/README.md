colormunki_photo Cookbook
==========================
Installs IHS Eviews

Requirements
------------
#### platform
- `Windows` - 8 or greater.

Attributes
----------

Usage
-----
#### ihs_eviews::win

Just include `ihs_eviews::win` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[ihs_eviews::win]"
  ]
}
```
